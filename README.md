# 🛒 IoT-Based Smart Trolley System Using NodeMCU ESP8266 and RFID for Automated Billing

##  Overview

The IoT Based Smart Trolley System is designed to automate the shopping and billing process using RFID technology and the NodeMCU ESP8266. The system detects products automatically, displays product details on an LCD screen, and updates the total bill in real time, reducing billing time and improving the shopping experience.

---

##  Features

- RFID-based product detection
- Automatic billing system
- Real-time LCD display
- Wi-Fi connectivity using NodeMCU ESP8266
- Faster checkout process
- Reduced manual billing errors

---

##  Hardware Used

- NodeMCU ESP8266
- RFID RC522 Reader
- RFID Tags
- I2C LCD Display (16×2)
- Jumper Wires
- Breadboard
- Power Supply

---

##  Software Used

- Arduino IDE
- Embedded C
- ESP8266 Library
- MFRC522 Library
- LiquidCrystal I2C Library

---

##  Working

1. Scan the RFID tag attached to the product.
2. NodeMCU identifies the product.
3. Product details are displayed on the LCD.
4. The total bill is updated automatically.
5. Billing information can be viewed through the web interface.
